# Play Store Data Analysis Dashboard

An interactive Power BI dashboard that provides insights into Google Play Store apps by analyzing categories, ratings, install patterns, and app types (Free vs Paid).

### 1. Project Overview

* This dashboard helps users explore key characteristics of Play Store applications and understand how different app categories perform in terms of quantity, user ratings, monetization, and total installs.
* It is designed for product managers, app developers, analysts, and anyone interested in mobile app market behavior.

### 2. Dashboard Purpose

* To compare app performance across Play Store categories.
* To analyze average ratings and identify category strengths.
* To understand the distribution of Free vs Paid apps.
* To evaluate install trends and top-performing apps.
* To explore how ratings correlate with install counts.

### 3. Tech Stack

* Power BI Desktop – Dashboard creation & interactive visuals
* Power Query – Data cleaning, transformation, and preparation
* DAX – Measures for KPIs such as total apps, ratings, paid app %, installs
* Data Modeling – Relationships enabling cross-filtering by category, rating, and type

### 4. Data Source

Dataset: Google Play Store Dataset (Kaggle)

Fields used in this dashboard:
* App Name
* Category
* Rating
* Installs
* Type (Free/Paid)

### 5. Key Features
~ KPI Metrics

* 10.8K Total Apps
* 3.62 Average Rating
* 7.4% Paid Apps
* 167.6B Total Installs

### 6. Visuals Included

* Number of Apps by Category
Shows which categories contain the most apps.
* Average Rating by Category
Compares user satisfaction across categories.
* Free vs Paid Apps by Category
Displays monetization distribution for each category.
* Top Apps by Installs
Table listing high-install apps like WhatsApp, Instagram, Subway Surfers, etc.
* Installs vs Rating
Bubble chart showing how user ratings relate to download volume.

### 7. Filters Available

* App Type (Free / Paid)
* Rating range slider
* Category multi-select slicer
* These allow flexible, dynamic exploration of the dataset.

### 8. Insights From the Dashboard

* Most apps on the Play Store are free, with paid apps being a small minority.
* Certain categories (e.g., Tools, Family, Games) dominate in app count.
* Average ratings vary significantly between categories.
* A few apps contribute disproportionately to total installs.
* Higher-rated apps often achieve higher install counts.
